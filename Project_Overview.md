# Netflix Movies & TV Shows Analysis

## Objective
Analyze Netflix content to understand trends, genres, ratings,
and regional distribution.

## Dataset
Netflix Movies and TV Shows - Kaggle

## Tasks Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualizations
- Advanced Analysis (Optional)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate Netflix content
- Content growth increased after 2015
- TV-MA is the most common rating
- USA and India lead content production
